# Linux-Configuartion-files
Contains various customizations of open-source User Interface tools like rofi, waybar, hyprland, etc. Customizations intended for Asus ZenBook Pro Duo. Drop in the .config folder located in your home directory.

# Hypr customization
 ## Hyprland customization
 hyprland.lua contains lua-compliant config file, which is still under development. hhyprland.conf contains full customizations. Includes notifications as OSD for most system changes.
 - Custom keybinds
 - Lidswitch behaviour (default set to hibernate)
 - Fine Brightness and Volume Control
 - Power profiles switch using Super + B
 - Display resolution switch using Super + R
 - Do Not Disturb (Super + D), Hypridle and Lid Behaviour pause (Super + SHIFT + I)

  ## Hyprlock customization
  Uses a UI similar to the GNOME lockscreen. Shows media info on the ScreenPad.

# Waybar customization
  ## Clock and Music (CnM)
  Displays the time and music with artist persistently, eliminating the need for a persistent taskbar taking up precious space. Can be toggled using Super + P.

  ## Status Bar
  Shows up when using the Super key, to show device stats like battery, wifi, volume, brightness, etc.

# Rofi customization
  Rofi launches full screen, with a modern UI including rounded corners and icons, and a translucent background. 

# Rofimoji customization
  A drop in replacement for the emoji picker. Uses Rofimoji.
